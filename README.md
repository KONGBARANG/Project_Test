## 💝 Customization

### Adding More Photos
1. Add your photos to the `photo/` folder
2. Update the HTML in `index.html` to include new photo sections
3. Update CSS animations if needed

### Adding More Videos
1. Add videos to the `video/` folder
2. Update the video sections in `index.html`
3. Ensure videos have appropriate posters

### Customizing Messages
- Edit the love messages in `index.html`
- Modify the timeline events
- Update the `LOVE_MESSAGES` array in `back.js` for modal content

### Changing Colors/Themes
- Modify the CSS variables and gradients in `css.css`
- Update the color scheme to match your preferences
